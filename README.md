# pashu_ahar

Pashu_Ahar is a smart livestock management application designed to support farmers in providing proper nutrition and care for their animals. The app helps users track cattle feed, monitor animal health, manage feeding schedules, and receive recommendations for balanced nutrition to improve livestock productivity and overall farm efficiency.
